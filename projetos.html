Projeto RIBELIA

O Projeto Ribelia é uma loja virtual fictícia desenvolvida com foco em front-end, onde apliquei conhecimentos de HTML, CSS e JavaScript para construir um catálogo de produtos com sistema de filtro por categoria e preço. O principal objetivo do projeto foi simular a navegação de uma usuária em um ambiente real de compras, permitindo aplicar filtros personalizados e ver apenas os produtos que se encaixam em sua busca. Trabalhei com manipulação de DOM, atributos `data-*`, classes dinâmicas e responsividade, além de utilizar boas práticas de organização visual e estrutura de código. Esse projeto representa um marco na minha evolução como futura desenvolvedora, pois foi o primeiro onde uni programação, lógica e design de forma integrada.

Projeto Lotus

O Projeto Lotus é uma vitrine digital inspirada em marcas de cosméticos e spa, desenvolvida com foco em HTML, CSS e organização visual refinada. Com uma proposta de estética limpa e elegante, o site apresenta produtos com imagens de alta qualidade e descrição clara, simulando uma experiência real de navegação por um catálogo online. A estrutura utiliza conceitos de semântica, boas práticas de estilização e layout com Flexbox para organizar os cards de forma responsiva. Além disso, o cabeçalho conta com logotipo e menu hamburguer, reforçando o aprendizado em design de interface e usabilidade. O Projeto Lotus representa minha evolução em construir sites bonitos, funcionais e voltados para o usuário.

Projeto Gabriela Caluz

O Projeto Gabriela Caluz é um portfólio pessoal desenvolvido com foco na identidade e trajetória da desenvolvedora. Totalmente construído com HTML e CSS, ele foi pensado para transmitir profissionalismo, organização e estilo próprio. A interface conta com um layout escuro moderno, destaque para a foto de apresentação em formato circular com borda personalizada e navegação clara entre seções como “Sobre mim”, “Projetos” e “Contatos”. Este projeto reforça minha evolução no domínio de Flexbox, estilização de textos e imagens, semântica HTML e boas práticas de design. Além de ser um espaço para apresentar quem sou, é também uma demonstração prática do que posso construir enquanto desenvolvedora front-end.
